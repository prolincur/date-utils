# date-utils
A collection utility methods for parsing (including excel serial date), formatting and manipulating dates.


#### Install
```
yarn add date-utils
```
or
```
npm i date-utils
```

#### Usage
```
import { DateUtil } from 'date-utils-lite' // or ''date-utils'

```
